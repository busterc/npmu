# npmu [![NPM version][npm-image]][npm-url]

> cli for uninstalling dependencies with a checkbox interface, for npm or pnpm

## Installation

```sh
$ npm install npmu --global
```

## Usage

```sh
$ npmu

# or, if you use pnpm
$ pnpmu

```
